## ğŸš€ How to Run Locally

Since this project uses JavaScript `fetch` to load the header/footer, you cannot simply open the `index.html` file. You must run a local server.

**Prerequisites:**
* You need **Python** installed (which, as a Data Scientist, you definitely have!).

**Steps:**

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/TheSaifHub/Saif-Ibrahim-Portfolio.git](https://github.com/TheSaifHub/Saif-Ibrahim-Portfolio.git)
    cd portfolio
    ```

2.  **Start a Local Server (Python)**
    ```bash
    # If using Python 3
    python -m http.server
    ```

3.  **View in Browser**
    Open your browser and navigate to:
    `http://localhost:8000`
